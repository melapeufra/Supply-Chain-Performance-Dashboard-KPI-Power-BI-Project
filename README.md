# Supply Chain Performance Dashboard | Power BI Project

## Overview

This project presents an interactive **Supply Chain Performance Dashboard** developed using **Microsoft Power BI** to support operational monitoring, KPI tracking, and data-driven decision-making.

The dashboard consolidates supply chain and operational data into a centralized analytics solution, enabling users to analyze:

* Performance KPIs
* Operational efficiency
* Program/site comparison
* Monthly trends
* Root cause analysis
* Weighted adherence metrics

This project demonstrates practical skills in:

* Business Intelligence (BI)
* Data Analytics
* Supply Chain Analytics
* KPI Monitoring
* Data Visualization
* ETL & Data Cleaning
* Dashboard Design
* Performance Reporting

---

# Objectives

The main objective of this project is to transform raw operational data into actionable business insights through:

* Data cleaning and preparation
* KPI standardization
* Interactive visualizations
* Performance analysis
* Decision-support reporting

The dashboard was designed to simulate real-world industrial and supply chain monitoring environments.
